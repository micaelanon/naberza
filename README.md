# naBerza

Sistema personal de tareas, recordatorios y citas con dashboard cuidado y recordatorios persistentes.

## Objetivo de la v1

Entregar una base mínima pero útil para:
- ver tareas generales en un dashboard bonito
- distinguir tareas persistentes de tareas normales
- marcar tareas como hechas
- preparar el camino para recordatorios diarios por canal
- desplegar en `pre` y `pro`

## Estructura del repo

```text
naberza/
├── .github/
├── code/
└── README.md
```

Todo el código vive dentro de `code/`.

## Estado actual

V1 básica visual del dashboard y estructura de proyecto.
Los recordatorios automáticos reales y la persistencia backend quedan para la siguiente iteración.
